# Vim Configuration

Optimised for web development.

## Usage

1. Clone the repository

```bash
git clone --recursive https://github.com/tlvince/vim-config.git $HOME/.vim
```

2. Compile YouCompleteMe

```bash
cd $HOME/.vim
make build-ycm
```
